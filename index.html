    Git from SVN clone and sync recursive bash script

    Project maintained by Great-Antique Hosted on GitHub Pages — Theme by mattgraham

    Bash script for recursive Git<-Svn sync

    Commands

    • clone svnRepoUrl [authorsFilePath]
    • rclone svnBaseUrl gitReposPath [authorsFilePath]
    • sync gitRepoPath
    • rsync syncDir

    Note, that it normally works only when you have enabled http-bulk-updates option in ~/.subversion/servers file

    [global]
http-bulk-updates=on
